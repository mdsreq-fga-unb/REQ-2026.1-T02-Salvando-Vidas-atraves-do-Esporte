# US-10 — Correção de Registro de Presença

!!! quote "História de Usuário"
    > *"Como **Voluntário**, quero corrigir um registro de presença lançado, para manter o histórico de frequência em conformidade."*
    > 
    > **Requisito Relacionado:** [RF14](../../Visão%20do%20Produto%20e%20Projeto/requisitosDeSoftware.md#rf14)

---

### Rota no App

!!! info "Navegação passo a passo"
    - **Pela tela de Frequência:** `Menu Principal` ➔ `Turmas` ➔ Selecionar Card da Turma ➔ Botão **"Registrar Frequência"** ➔ Clicar no Ícone de Calendário ➔ Selecionar Data Anterior ➔ Alterar Presenças ➔ Botão **"Registrar Frequência"** ➔ Modal *Confirmação* ➔ Botão **"Confirmar"**
    - **Pelo Histórico:** `Menu Principal` ➔ `Turmas` ➔ Selecionar Card da Turma ➔ Menu Opções (⋮) ➔ Opção **"Histórico de Frequência"**

---

### Critérios de Aceitação

- [x] O status de presença de um registro já lançado pode ser alterado pelo voluntário.
- [x] O sistema deve exibir ao voluntário o status de presença atualizado imediatamente após a confirmação da alteração.
- [x] A correção deve ser refletida imediatamente no histórico do aluno.
