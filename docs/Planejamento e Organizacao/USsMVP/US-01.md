# US-01 — Cadastro de Aluno

!!! quote "História de Usuário"
    > *"Como **Coordenador**, quero registrar dados de alunos, para reduzir a dependência de fichas de papel."*
    > 
    > **Requisito Relacionado:** [RF01](../../Visão%20do%20Produto%20e%20Projeto/requisitosDeSoftware.md#rf01)

---

### Rota no App

!!! info "Navegação passo a passo"
    - **Aluno menor de 18 anos:** `Menu Principal` ➔ `Início` ➔ Botão **"Cadastrar Aluno"** ➔ Preencher Aba *Dados Básicos* ➔ Botão **"Avançar"** ➔ Preencher Aba *Dados Médicos* ➔ Botão **"Avançar"** ➔ Preencher Aba *Dados do Responsável* ➔ Botão **"Cadastrar"** ➔ Modal *Confirmar Cadastro* ➔ Botão **"Confirmar"**
    - **Aluno maior ou igual a 18 anos:** `Menu Principal` ➔ `Início` ➔ Botão **"Cadastrar Aluno"** ➔ Preencher Aba *Dados Básicos* ➔ Botão **"Avançar"** ➔ Preencher Aba *Dados Médicos* ➔ Botão **"Cadastrar"** ➔ Modal *Confirmar Cadastro* ➔ Botão **"Confirmar"**

---

### Critérios de Aceitação

- [x] **Validação de Campos:** O sistema deve validar o preenchimento dos campos obrigatórios antes de concluir o cadastro do aluno.
- [x] **Dados do Responsável:** Quando o aluno for menor de idade, o sistema deve solicitar o preenchimento dos dados do responsável.
- [x] **Localização na Busca:** Após o cadastro, o aluno deve ser localizado pela funcionalidade de busca utilizando seu nome.
