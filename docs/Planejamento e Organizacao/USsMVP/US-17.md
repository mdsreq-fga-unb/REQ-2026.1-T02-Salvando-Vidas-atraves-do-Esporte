# US-17 — Registro de Devolução de Kimono

!!! info "📍 Rota no App"
    Menu Principal > Estoque > Empréstimos > Registrar Devolução

---

## História de Usuário

Como **Coordenador**, quero registrar a devolução de um kimono, para reintegrá-lo ao estoque disponível para empréstimo.

**Requisito relacionado:** [RF22](../Visão%20do%20Produto%20e%20Projeto/requisitosDeSoftware.md#rf22)

---

## Critérios de Aceitação

* O sistema deve reintegrar automaticamente o kimono ao estoque disponível após o registro da devolução.
* O sistema não deve permitir o registro da devolução de um kimono que não possua empréstimo ativo.
* O sistema deve exibir a lista de alunos com empréstimos ativos para seleção durante o processo de devolução.
