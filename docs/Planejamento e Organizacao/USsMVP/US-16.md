# US-16 — Registro de Empréstimo de Kimono

!!! quote "História de Usuário"
    > *"Como **Coordenador**, quero registrar o empréstimo de um kimono a um aluno, para saber onde cada uniforme está a qualquer momento."*
    > 
    > **Requisito Relacionado:** [RF21](../../Visão%20do%20Produto%20e%20Projeto/requisitosDeSoftware.md#rf21)

---

### Rota no App

!!! info "Navegação passo a passo"
    - `Menu Principal` ➔ `Inventário` ➔ Botão **"Emprestar Kimono"** ➔ Selecionar Aluno ➔ Escolher Kimono no Estoque ➔ Botão **"Avançar"** ➔ Modal *Confirmação* ➔ Botão **"Confirmar"**

---

### Critérios de Aceitação

- [x] O sistema deve vincular o empréstimo do kimono a um aluno com cadastro ativo e removê-lo automaticamente do estoque disponível.
- [x] O sistema não deve permitir o empréstimo de kimonos indisponíveis no estoque.
- [x] O sistema deve permitir apenas um empréstimo ativo de kimono por aluno.
