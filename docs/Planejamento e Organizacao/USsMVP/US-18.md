# US-18 — Aniversariantes da Semana no Dashboard

!!! quote "História de Usuário"
    > *"Como **Voluntário**, quero ver no dashboard os aniversariantes da semana, para que a equipe celebre com os alunos no treino."*
    > 
    > **Requisito Relacionado:** [RF29](../../Visão%20do%20Produto%20e%20Projeto/requisitosDeSoftware.md#rf29)

---

### Rota no App

!!! info "Navegação passo a passo"
    - `Menu Principal` ➔ `Início` ➔ Card *Próximos aniversariantes* ➔ Clicar sobre o item/nome do aluno na lista ➔ Modal *Data do aniversário*

---

### Critérios de Aceitação

- [x] O sistema deve exibir automaticamente a lista de aniversariantes da semana no dashboard após a autenticação do usuário.
- [x] A lista de aniversariantes deve apresentar claramente o nome do aluno e a quantidade de dias restantes até o aniversário.
- [x] Caso não existam aniversariantes na semana corrente, o sistema deve exibir uma mensagem informativa ao usuário.
