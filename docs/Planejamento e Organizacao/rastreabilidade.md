# Rastreabilidade

Conforme sugerido pelo professor, este artefato apresenta a representação da árvore de rastreabilidade, contemplando as Histórias de Usuário (User Stories). A árvore demonstra visualmente quais histórias estão implementadas e quais permanecem fora do escopo do MVP.

Além disso, o artefato evidencia o nível de completude dos Critérios de Produto (CP) e dos Objetivos Específicos (OE), utilizando a seguinte convenção de cores: **verde** para itens completamente implementados, **amarelo** para itens parcialmente implementados e **cinza** para aqueles que não fazem parte do escopo do MVP.

Os Requisitos Não Funcionais (RNFs) transversais estão representados no canto inferior esquerdo da árvore, juntamente com a legenda utilizada para facilitar sua interpretação.

Como imagens não suportam hyperlinks para páginas específicas, foi disponibilizada, logo abaixo da figura, uma tabela contendo os respectivos links para cada artefato referenciado, além do acesso ao projeto no Figma, ferramenta utilizada para o desenvolvimento da árvore de rastreabilidade, preservando todos os hyperlinks originalmente inseridos.
