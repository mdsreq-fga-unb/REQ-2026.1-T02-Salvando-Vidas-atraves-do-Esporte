- **Stakeholders**:
    - José Lucas (Representante do Projeto Social)
    - Aline (Voluntária do Projeto Social)
    - João (Voluntário do Projeto Social)

### Resumo

Nesta reunião, foram realizadas as validações da primeira versão do aplicativo — que contava apenas com a tela de login e o cadastro de voluntários — e dos protótipos de baixa fidelidade da página inicial (dashboard) e do cadastro de alunos. 

Em relação à primeira versão do aplicativo, os voluntários e o representante aprovaram o aspecto visual, cientes de que ainda se tratava de uma versão inicial. No entanto, solicitaram a adição de um campo para CPF no cadastro de voluntários.

Os stakeholders também tiveram uma impressão positiva dos protótipos apresentados, mas reforçaram a necessidade de acrescentar uma aba para coletar informações médicas básicas, que não estava presente na versão demonstrada.

## R7 - Reunião 03/06 - Priorização do MVP: priorização das User Stories com a técnica do MoSCoW { #reuniao-r7 }

<iframe src="https://drive.google.com/file/d/1ANG92X-n4NnMNAb-J_wUUmgj6HJkW9p_/preview" width="640" height="480"></iframe>

### Participantes

- **Equipe**:
    - Giovani - 241032500
    - José Augusto - 231026429
    - Júlia - 241036142
    - Lucas - 241011386
    - Pedro Lucas - 241025710
    - Pedro Ramos - 222031680
- **Stakeholders**:
    - José Lucas (Representante do Projeto Social)
    - Aline (Voluntária do Projeto Social)
    - Leonardo (Voluntário do Projeto Social)

### Resumo

A reunião correspondeu à **Priorização do MVP**, com foco na priorização das histórias de usuário utilizando a técnica **MoSCoW**.

A dinâmica foi conduzida com apoio do **Miro**, utilizando cores para tornar a priorização mais visual e acessível aos clientes. A equipe e os clientes passaram por cada história de usuário atribuindo notas e categorias MoSCoW.

Durante a sessão, os clientes **reduziram a prioridade da criação de turmas para Should**, justificando que as turmas permanecem as mesmas há muito tempo e, portanto, essa funcionalidade não é prioritária no momento.

### Resumo
